# 个人博客项目页面结构图

## 1. 页面总览

```text
my-blog
├─ 首页 /
├─ 文章列表 /posts
├─ 文章详情 /posts/[slug]
├─ 分类页 /categories
├─ 分类详情 /categories/[slug]
├─ 标签页 /tags
├─ 标签详情 /tags/[slug]
├─ 归档页 /archives
├─ 项目页 /projects
├─ 关于页 /about
└─ 404 页面
```

## 2. 页面层级关系

```text
首页
├─ 最新文章
│  └─ 进入文章详情页
├─ 推荐内容
│  └─ 进入文章详情页
├─ 标签入口
│  └─ 进入标签详情页
├─ 项目入口
│  └─ 进入项目页
└─ 关于入口
   └─ 进入关于页
```

```text
文章列表页
├─ 文章卡片
│  ├─ 标题
│  ├─ 摘要
│  ├─ 发布时间
│  └─ 标签
└─ 点击卡片进入文章详情页
```

```text
文章详情页
├─ 文章头图/标题区
├─ 元信息
│  ├─ 发布时间
│  ├─ 分类
│  └─ 标签
├─ 目录
├─ 正文
├─ 上一篇/下一篇
└─ 评论区
```

## 3. 通用布局结构

```text
全站布局
├─ Header
│  ├─ Logo/站点名
│  ├─ 导航菜单
│  └─ 主题切换或搜索入口
├─ Main
│  └─ 当前页面内容
└─ Footer
   ├─ 版权信息
   ├─ 社交链接
   └─ 订阅或备案信息
```

## 4. 首页模块结构

```text
首页
├─ Hero 主视觉区
│  ├─ 博客名称
│  ├─ 一句介绍
│  └─ CTA 按钮
├─ 最新文章区
├─ 分类/标签快捷入口
├─ 项目展示区
└─ 页脚
```

## 5. 文章详情页模块结构

```text
文章详情页
├─ 标题区
├─ 文章元信息
├─ 目录导航
├─ 正文内容
├─ 相关文章推荐
├─ 上一篇/下一篇
└─ 评论区
```

## 6. 后续可扩展页面

```text
扩展页面
├─ 搜索页 /search
├─ 订阅成功页 /subscribe-success
├─ 站点地图页 /sitemap
└─ 友链页 /friends
```
