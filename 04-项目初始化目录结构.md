## 2. 目录说明

### `public/`

- 放置静态资源
- 比如头像、文章封面、图标、favicon

### `src/app/`

- 页面路由目录
- 对应首页、文章页、关于页等

### `src/components/`

- 放通用组件
- 例如导航栏、页脚、文章卡片、按钮

### `src/content/posts/`

- 存放 Markdown 或 MDX 文章文件

### `src/lib/`

- 放业务工具函数
- 如读取文章、解析 Markdown、生成 SEO 数据

### `src/types/`

- 放 TypeScript 类型定义

### `src/config/`

- 放站点配置
- 例如站点标题、描述、社交链接

## 3. 文章文件建议格式

```text
src/content/posts/
├─ my-first-post.md
├─ nextjs-notes.md
└─ yearly-review-2026.md
```

每篇文章可包含 frontmatter：

```md
---
title: "我的第一篇文章"
date: "2026-04-28"
description: "这是一篇用于测试博客系统的文章"
tags: ["日记", "博客"]
category: "随笔"
draft: false
cover: "/images/post-cover.jpg"
---
```

## 6. 初始化建议

如果后续正式开始开发，可以先完成下面这些内容：

1. 初始化 Next.js + TypeScript 项目
2. 配置 Tailwind CSS
3. 创建基础布局
4. 加入 Markdown 文章读取逻辑
5. 完成首页和文章详情页
