# my-blog

一个使用 Next.js、TypeScript、Tailwind CSS 和 Markdown 搭建的个人博客初始框架。

## 启动

```bash
npm install
npm run dev
```

访问 `http://localhost:3000`。

## 当前包含

- 首页
- 关于页
- 文章列表页
- 文章详情页
- Markdown 文章读取
- 基础站点配置

## 文章目录

文章放在 `src/content/posts/` 下，使用 Markdown 编写。
